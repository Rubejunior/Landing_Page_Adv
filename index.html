Mas é um copia e cola, basta substituir. #Acima tem o script: fetch('https://api.ipify.org?format=json') veja a explicação e se é útil para você, caso não for, exclua. Ele tem a função de salvar os IP's dos usuários que acessam o site, é preciso ter um servidor para salvar os dados. É uma aplicação útil para proteção do site, podendo bloquear IP's suspeitos, e mais ainda para campanhas ads, IP'S com mais der uma certa quantidade de acessos eu bloqueio nas campanhas do GOOGLE e FACEBOOK para evitar cliques de ROBOS e demais outros casos não necessários. Cada caso é um caso, essa é uma pagina de venda com principal objetivo de capturar o cliente no primeiro acesso ou no máximo segundo (eu nesse caso bloqueioo todos ip's que acessaram o site pelo menos uma vez em todas as campanhas ads)
